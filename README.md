# 🧰 SaaS Marketing Stack — Tools, Channels & Metrics by Stage

Stop guessing which marketing tools to buy. This is a curated stack guide for SaaS startups that maps **tools** (analytics, attribution, email, CRM), **channels** (what works at $0, $10K, and $100K/month spend), a **content engine** (SEO, blog, comparison pages, distribution), and **paid acquisition** (when to start, how to test, when to kill) — all indexed by growth stage and budget.

## Install first, browse later

```bash
npx skills add Gingiris-1031/saas-marketing-playbook
# or
clawhub install saas-marketing-playbook
```

## Four-language digest

- 🇺🇸 Know exactly which tool and channel to adopt at your stage — and which ones to ignore until later.
- 🇨🇳 一份按阶段和预算索引的 SaaS 营销栈指南：什么阶段买什么工具、$0/$10K/$100K 月预算各投什么渠道、内容引擎怎么搭、付费投放何时启动何时砍。
- 🇯🇵 成長ステージと予算で引けるSaaSマーケティングスタックのガイド。どの段階でどのツール・チャネルを採用し、何を後回しにすべきか。
- 🇰🇷 성장 단계와 예산으로 찾아보는 SaaS 마케팅 스택 가이드. 어느 단계에 어떤 도구와 채널을 도입하고 무엇을 미뤄야 하는지.
